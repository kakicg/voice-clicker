# VoiceClicker

VoiceClicker は、あらかじめ登録した画面上の位置を、声で「はい」と言ったときに左クリックする macOS メニューバー常駐アプリです。

Dock には表示されず、起動中はメニューバーにマイクのアイコンが表示されます。

## 必要な環境

- macOS
- Xcode
- マイク入力

このプロジェクトは Xcode プロジェクトとして管理されています。

```sh
VoiceClicker.xcodeproj
```

## インストール

### Xcode からインストールする

1. Xcode で `VoiceClicker.xcodeproj` を開きます。
2. スキーム `VoiceClicker` を選びます。
3. 必要に応じて Signing & Capabilities で Team を設定します。
4. `Product` > `Archive`、または `Product` > `Build` でアプリをビルドします。
5. 生成された `VoiceClicker.app` を `/Applications` にコピーします。
6. `/Applications/VoiceClicker.app` を一度起動します。

`VoiceClicker.app` は Xcode の一時的なビルド場所からではなく、`/Applications` に置いたものを使ってください。ログイン時起動を安定して動かすためです。

### コマンドラインでビルドする

次のコマンドで Release ビルドできます。

```sh
xcodebuild \
  -project VoiceClicker.xcodeproj \
  -scheme VoiceClicker \
  -configuration Release \
  -derivedDataPath build \
  build
```

ビルド後、生成されたアプリを `/Applications` にコピーします。

```sh
cp -R build/Build/Products/Release/VoiceClicker.app /Applications/
```

## 初回起動時の環境設定

初回起動時に、macOS の権限許可が必要です。

### マイク

「はい」という声を検知するために使います。

許可できなかった場合は、次の場所で有効にしてください。

```text
システム設定 > プライバシーとセキュリティ > マイク
```

### 音声認識

日本語の音声コマンドを認識するために使います。

許可できなかった場合は、次の場所で有効にしてください。

```text
システム設定 > プライバシーとセキュリティ > 音声認識
```

### アクセシビリティ

登録した位置をクリックするために使います。

許可できなかった場合は、次の場所で `VoiceClicker` を有効にしてください。

```text
システム設定 > プライバシーとセキュリティ > アクセシビリティ
```

## Mac ログイン時に自動起動する

1. `/Applications/VoiceClicker.app` を起動します。
2. メニューバーのマイクアイコンをクリックします。
3. `ログイン時に起動: オフ` をクリックして `オン` にします。

これで、次回以降 macOS にログインしたときに VoiceClicker が自動で起動し、メニューバーに常駐します。

これは Mac の電源投入直後ではなく、ユーザーが macOS にログインした後に起動する設定です。

## 基本操作

### クリック位置を登録する

1. メニューバーのマイクアイコンをクリックします。
2. `クリック位置を記録...` を選びます。
3. 画面が少し暗くなったら、音声でクリックしたい場所を実際にクリックします。
4. 登録できると、メニューに `登録位置: (x, y)` が表示されます。

登録画面は `Esc` キーでキャンセルできます。

### 音声でクリックする

クリック位置を登録した状態で、マイクに向かって「はい」と言います。

VoiceClicker は登録済みの位置にマウスカーソルを一瞬移動し、左クリックしてから元の位置に戻します。

### 登録位置をリセットする

メニューバーのマイクアイコンから `位置をリセット` を選びます。

登録位置がない状態では、「はい」と言ってもクリックは実行されません。

### 無操作リセットを設定する

メニューバーのマイクアイコンから `無操作リセット` を開き、次のいずれかを選びます。

- `15分`
- `30分`
- `60分`
- `120分`
- `なし`

初期設定は `30分` です。

無操作リセットが有効な場合、一定時間クリック操作がないと登録位置が自動で消えます。

### アプリを終了する

メニューバーのマイクアイコンから `終了` を選びます。

## 注意点

- 登録されるのは画面上の座標です。ウィンドウを移動すると、意図しない場所をクリックすることがあります。
- 外部ディスプレイの接続状態や解像度を変えた場合は、クリック位置を登録し直してください。
- 音声認識は日本語 `ja-JP` で動作します。検知する言葉は「はい」です。
- クリックが動かない場合は、アクセシビリティ権限を確認してください。
- 音声に反応しない場合は、マイク権限と音声認識権限を確認してください。
